== Description ==
&#x202b;این افزونه اطلاعات لیست تماس سرویس تلفنچی را در پنل وردپرس شما نمایش می دهد.
&#x202b;هدف این پلاگین با دسترسی به وب سرویس برای دریافت لیست تماس ها است.
## نیازمندی ها
* وردپرس نسخه ۵.۲.۰ به بالا
## نحوه استفاده
ابتدا پلاگین را فعالسازی کرده و سپس توکن وب سرویس و شناسه سرویس را در بخش تنظیمات وارد کنید.
== Installation ==

1. &#x202b;کافیست پلاگین را داخل پوشه `/wp-content/plugins/` کپی کنید و یا آن را از انباره پلاگین های وردپرس به صورت مستقیم نصب نمایید.
2. &#x202b;سپس پلاگین را از قسمت افزونه های داشبورد خودتان فعال نمایید.
